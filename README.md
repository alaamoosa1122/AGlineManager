# AGline Manager

Full stack management system for AGline (Abaya Management).

## Features
- Dashboard for sales and orders overview.
- Customer management.
- Order tracking and status updates.
- Design management with cost and price tracking.
- Role-based access (Admin/User).

## Tech Stack
- **Frontend**: React, Vite, CSS.
- **Backend**: Node.js, Express.
- **Database**: MongoDB.

## Deployment
- **Backend**: Render
- **Frontend**: Vercel / Render

## Local Setup
1. Clone the repository.
2. Run `npm run install-all` to install dependencies for both frontend and backend.
3. Create `.env` files in `backend/` and `frontend/` with necessary variables.
4. Run `npm run dev` to start the development server.
